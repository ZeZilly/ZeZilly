## About

I work at the intersection of AI-assisted execution, multilingual operations, and agentic system design.

My strength is not writing every line of code myself — it is understanding how systems should work, what needs to be built, where each part belongs, and how AI tools can be guided to turn ideas into structured, production-ready outcomes.

I specialize in: human-in-the-loop pipelines, prompt architecture, multilingual QA, content safety workflows, and orchestrating AI agents that operate with clear boundaries and measurable outputs.

> Research-first, supervised paper trading system — FastAPI · MCP · N8N · 5 connectors
